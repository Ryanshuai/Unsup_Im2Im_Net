# Unsupervised_Image_translation
变量名规则：

变量名      | 含义

_           | 下划线 之前是结构位置，之后是来源位置

A，B        | A域， B域  

C，D        | B_A域， A_B域

S，s        | 共享网络或参数  

R           | 重现的图片 

y           | discriminator的输出，正确的概率

pre         | AB单独的网络 

end         | AB单独的网络

mu          | 均值   

sigma       | 方差
